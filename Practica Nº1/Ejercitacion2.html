Ejercicitacion 1

2.a) El elemento de comentario puede ser colocado en cualquier parte del codigo, no afecta su fincionalidad ya que no es ejecutado.

2.b) El Div es utilizado en el body del documento. Lo que genera este div en particular es en proveer el atributo id el cual es "bloque1" a su contenido.
No es obligatorio usar el atributo id.

2.c) El img indica un elemento de imagen el cual es utilizado en el body del documento.
En este caso contiene atributos de src con valor "" (vacio), alt con valor "lugar imagen", id con valor "im1", name con valor "im1", width con valor "32", height con valor "32" y longdesc con valor "detalles.htm".
El atributo name no debe formar parte del elemento.
El unico atributo obligatorio es el src.

2.d) El elemento meta provee metadatos que no pueden ser expresados con otros elementos del encabezado (head), como title, base, link, style y script. Simpre va ubicado en el head del documento.
En el primer caso contiene atributos de name con valor "keywords", lang con valor "es" y content con valor "casa, compra, venta, alquiler".
En el segundo contiene atributos de http-equiv con valor "expires" y content con valor "16-Sep-2019 7:49PM".
No son obligatorios ninguno de los atributos.

2.e) El elemento a es utilizado en el body del documento. Este representa un enlace o hypervinculo que se usa para acceder de una pagina a otra.
En nuestro caso contiene el atributo href con el valor "http://www.e--style.com.ar/resumen.html", type con valor "text/html", hreflang con valor "es", charset con valor "utf-8" y rel con valor "help".
Ningun atributo es obligatorio.

2.f) El elemento table define una tabla en HTML.
En este caso contiene los atributos width con valor "200" y summary con valor "Datos correspondientes al ejercicio vencido"
El elemento caption define el titulo de la tabla.
En este caso contien el atributo align con valor "top"
El elemento tr define una nueva fila en la tabla.
El elemento th define una cabecera de la tabla. En este caso con el atributo scope y valor "col" en algunos lo cual especifica encabezado de columnas y "row" en otro el cual especifica encabezado de fila.
El elemento td define una celda.
Ningun atributo es obligatorio.
